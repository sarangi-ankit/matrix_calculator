
# Matrix Calculator




## Installation

1.Clone the repository:

```bash
  git clone https://github.com/sarangi-ankit/matrix_calculator.git

```
2.Navigate to the project directory:

```bash
  cd your-repository


```
3.Install dependencies:
```bash
  npm install



```
## Usage

Run the development server::

```bash
  npm run dev


```
